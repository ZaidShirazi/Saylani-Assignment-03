html

HTML is a markup language used to structure the content of a webpage. It defines the elements and their hierarchy, such as headings, paragraphs, images, links, and lists.

css

CSS is a style sheet language used to control the presentation and visual appearance of HTML elements. It dictates aspects like colors, fonts, layout, spacing, and responsiveness.

Javascript

JavaScript is a programming language that adds interactivity and dynamic behavior to webpages. It enables features like animations, form validation, dynamic content updates, and user interaction responses.
